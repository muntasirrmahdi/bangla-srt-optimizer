# Contributing to Bangla SRT Optimizer

Thank you for your interest in improving the Bangla SRT Optimizer!

## How to Contribute
1. **Report Bugs**: Open an issue using the Bug Report template.
2. **Suggest Features**: Open an issue using the Feature Request template.
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature/fix.
   - Ensure your code follows the existing patterns.
   - Submit a PR with a clear description of changes.

## Code Standards
- Use modular structure (Logic in `core/`, UI in `app.py`).
- Maintain 100% timestamp accuracy.
- Document any new hardcoded rules in `rules_sample.json`.
